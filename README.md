# 고품질 음악 스트리밍 사이트

2023.04-2023.06


## 프로젝트 개요

고품질의 음악 스트리밍을 위해서만 많은 금액을 지불해야 하기 때문에 학생들을 위한 무료 고품질 음악 스트리밍 사이트를 제작하였습니다.

## 개발 환경

+ Node.js
+ 데이터베이스 : MySQL
